# XML Organigrama Empresa

Proyecto académico para el diseño, creación y validación de documentos XML basados en la estructura organizativa de una empresa.

## Contenido
- `docs/`: organigrama de la empresa en imagen
- `xml/`: documento XML
- `dtd/`: archivo DTD para validación

## Objetivo
Representar la jerarquía de una empresa mediante XML y validar su estructura con una DTD.

## Estructura trabajada
La empresa **TechNova** se organiza en tres departamentos:
- Dirección
- Recursos Humanos
- Tecnología
